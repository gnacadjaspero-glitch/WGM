===========================================================================
      GUIDE D'INSTALLATION ET D'UTILISATION - WINNER GAME MANAGER (WGM)
===========================================================================

Ce guide vous explique comment mettre en service le système de A à Z.

---------------------------------------------------------------------------
1. PRÉREQUIS MATÉRIELS ET LOGICIELS
---------------------------------------------------------------------------
- Matériel : Un ESP32, des modules Relais (5V), un PC sous Windows.
- Logiciels (Développement) :
  * Flutter SDK (Version 3.10+) : Pour l'application.
  * Arduino IDE : Pour programmer le boîtier ESP32.
  * Inno Setup 6 : Pour créer l'installateur Windows (.exe).

---------------------------------------------------------------------------
2. CONFIGURATION DU BOÎTIER (HARDWARE)
---------------------------------------------------------------------------
1. Ouvrez le fichier 'hardware/esp32_controller.ino' dans Arduino IDE.
2. Installez les bibliothèques nécessaires si demandées (WiFi, WebServer).
3. Connectez votre ESP32 à votre PC via USB.
4. Dans Arduino IDE, sélectionnez la carte "DOIT ESP32 DEVKIT V1".
5. Cliquez sur "Téléverser" (Upload).
   -> Une fois terminé, le boîtier émettra un réseau WiFi nommé "WGM".

---------------------------------------------------------------------------
3. AUTOMATISATION VIA 'build_project.bat' (MÉTHODE RAPIDE)
---------------------------------------------------------------------------
Le fichier 'build_project.bat' à la racine automatise tout le processus :

1. Double-cliquez sur 'build_project.bat'.
2. Menu des actions :
   - Option 1 : Générer l'APK Android (pour installation sur mobile/tablette).
   - Option 2 : Compiler les fichiers Windows Release (requis avant l'installateur).
   - Option 3 : Créer l'installateur EXE final (utilise Inno Setup automatiquement).
   - Option 4 : Pub Get (répare les dépendances si le projet ne se lance pas).
   - Option 5 : Clean (nettoie les fichiers temporaires en cas d'erreur).

---------------------------------------------------------------------------
4. CONFIGURATION MANUELLE (FLUTTER)
---------------------------------------------------------------------------
Si vous préférez utiliser la ligne de commande :
1. Installez les dépendances : 'flutter pub get'
2. Testez en direct : 'flutter run -d windows'
3. Compilez pour Windows : 'flutter build windows --release'
4. Compilez pour Android : 'flutter build apk --release'

---------------------------------------------------------------------------
5. MISE EN SERVICE (UTILISATION RÉELLE)
---------------------------------------------------------------------------
1. Alimentez le boîtier ESP32.
2. Sur votre PC Windows, connectez-vous au réseau WiFi "WGM".
   - Mot de passe WiFi : CheatWinner@360
3. Installez l'application via le fichier 'WinnerGameManager_Setup.exe'
   (disponible à la racine du dépôt pour un accès direct).
4. Identifiants de connexion :
   - Mot de passe ADMIN : ******************
5. Vérifiez l'icône de connexion (Haut-Droit) :
   - VERT : Communication établie avec le boîtier (IP: 192.168.100.1).
   - ROUGE : Problème de connexion WiFi ou IP.
6. Gérez vos postes en toute sécurité.

---------------------------------------------------------------------------
6. SÉCURITÉ ET AUDITS
---------------------------------------------------------------------------
- Logs Furtifs : L'activité est enregistrée dans des dossiers cachés pour
  éviter toute suppression par les gérants.
  * Windows : %AppData%/Roaming/Audit_Logs/
  * Android : Dossier .wgm_audit
- Sécurité Réseau : Clé de chiffrement unique entre l'App et l'ESP32.
